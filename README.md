# Library Management System (JDBC + MySQL)

A console-based Library Management System developed using Java, JDBC, and MySQL. The application allows users to manage books and library members through a menu-driven interface.

## Features

- Add, View, Search, and Delete Users
- Add, View, Search, and Delete Books
- Issue Books to Users
- Return Issued Books
- MySQL Database Integration using JDBC
- Menu-Driven Console Interface
- CRUD Operations using PreparedStatement

## Technologies Used

- Java
- JDBC
- MySQL
- VS Code

## Database Tables

### users

- user_id
- name

### books

- book_id
- title
- author
- available
- issued_to
